# Mechinet

**מערכת ניהול קבלה למכינות קדם-צבאיות**

פלטפורמת קוד פתוח בעברית לניהול תהליך הקבלה של מכינות. בונים טופס מועמדות מותאם אישית, מנהלים מאות מועמדים בדשבורד אחד, ומקבלים סיכומי AI לכל טופס — כשההחלטה הסופית תמיד נשארת אצל הצוות החינוכי.

---

## למה?

כל שנה מכינות קדם-צבאיות מקבלות מאות עד אלפי טפסי מועמדות. רובן מנהלות את התהליך באקסל, גוגל פורמס, וואטסאפ ומיילים — בלאגן שגורם לטעויות, מועמדים שאובדים, וזמן יקר של הצוות החינוכי שמתבזבז על אדמיניסטרציה.

Mechinet הוא ניסיון לפתור את זה — בלי להחליף את שיקול הדעת האנושי, אלא לתת כלים שמסדרים את התהליך.

---

## מה כלול ב-MVP

- ✅ הרשמה והתחברות לצוות
- ✅ בונה טפסים פשוט (טקסט, בחירה, קובץ)
- ✅ קישור ציבורי לטופס מועמדות
- ✅ דשבורד מועמדים עם חיפוש וסינון
- ✅ Pipeline שלבי קבלה (Drag & Drop)
- ✅ הערות וצוות
- ✅ סיכום AI אוטומטי לכל טופס
- ✅ ייצוא לאקסל

## מה לא כלול (ובכוונה)

- ❌ AI שמדרג או מציין מועמדים — ההחלטה תמיד אנושית
- ❌ דחייה אוטומטית — לעולם לא
- ❌ מערכת מורכבת של ראיונות
- ❌ מערכת תשלומים (כרגע)

---

## טכנולוגיות

- **Next.js 16** + **TypeScript** + **Tailwind CSS**
- **Supabase** (Postgres + Auth + Storage)
- **OpenAI GPT-4o-mini** (לסיכומים)
- **Vercel** (אחסון)

---

## איך מתקינים (למפתחים)

```bash
# Clone the repo
git clone https://github.com/Ran08-stack/mechinet.git
cd mechinet

# Install dependencies
npm install

# Copy env example and fill in your keys
cp .env.example .env.local

# Run dev server
npm run dev
```

פתח את `http://localhost:3000` בדפדפן.
